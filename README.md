# PeruCompras Scraper

Basically this scraper pulls the information from `https://www.perucompras.gob.pe/contrataciones/contrataciones-emergencia-covid19.php`

## Installation

```bash
npm install
```

## Usage

```python
npm start
```

Output is generated by `output.csv`
